# 一键生成小红书组图工具

这是一个简单易用的工具，可以根据输入的文章内容，自动生成适合小红书平台的组图。

## 功能特点

- 支持输入文章内容，自动解析并生成组图
- 生成的图片符合小红书3:4的尺寸要求
- 自动为每个内容块创建单独的图片
- 支持批量下载生成的组图
- 响应式设计，适配不同设备

## 如何使用

1. **启动应用**
   ```bash
   npm run dev
   ```
   然后在浏览器中访问 `http://localhost:3000`

2. **输入文章内容**
   在文本框中输入你的文章内容，可以使用Markdown格式的标题（以#开头）来分隔不同的内容块。

3. **生成组图**
   点击"生成组图"按钮，系统会自动解析文章内容并生成对应的组图。

4. **下载组图**
   生成完成后，点击"下载组图"按钮，系统会将所有生成的图片下载到你的设备。

## 技术实现

- **前端框架**：React
- **构建工具**：Vite
- **图片生成**：Canvas API
- **样式**：CSS

## 注意事项

1. 生成的图片数量限制在1-9张之间，这是小红书平台的限制。
2. 文章内容过长时，可能会导致生成的图片内容不完整，请适当控制每个内容块的长度。
3. 为了获得最佳效果，建议使用清晰的标题结构来组织文章内容。
