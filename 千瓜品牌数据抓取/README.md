# 千瓜数据抓取器

## 项目简介

千瓜数据抓取器是一个用于从千瓜数据平台抓取品牌在各社交媒体渠道数据的工具。该工具支持抖音、小红书、微博等多个平台，可根据品牌名称、渠道和日期范围抓取相关数据，并导出为Excel格式。

## 功能特点

- **多品牌支持**：内置古茗、库迪咖啡、星巴克、蜜雪冰城等品牌的ID映射
- **多渠道支持**：支持抖音、小红书、微博等多个社交媒体平台
- **灵活的日期范围**：可自定义抓取的开始和结束日期
- **Excel导出**：自动将抓取的数据导出为Excel格式，方便后续分析
- **命令行参数**：支持通过命令行参数指定品牌、渠道和日期范围

## 安装依赖

在运行前，需要安装以下Python包：

```bash
pip install pandas requests openpyxl
```

## 使用方法

### 配置Cookie

1. 登录千瓜数据平台
2. 获取Cookie并更新到 `qiangua_scraper.py` 中的 `CONFIG['cookie']`

### 命令行执行

```bash
# 抓取古茗在抖音的数据（2026-03-01至2026-03-15）
python3 qiangua_scraper.py --brand 古茗 --channel 抖音 --start-date 2026-03-01 --end-date 2026-03-15

# 抓取库迪咖啡在抖音的数据（2026-03-16至2026-03-31）
python3 qiangua_scraper.py --brand 库迪咖啡 --channel 抖音 --start-date 2026-03-16 --end-date 2026-03-31

# 抓取星巴克在小红书的数据（2026-03-01至2026-03-31）
python3 qiangua_scraper.py --brand 星巴克 --channel 小红书 --start-date 2026-03-01 --end-date 2026-03-31

# 抓取蜜雪冰城在微博的数据（2026-03-01至2026-03-31）
python3 qiangua_scraper.py --brand 蜜雪冰城 --channel 微博 --start-date 2026-03-01 --end-date 2026-03-31
```

### 支持的品牌

- 古茗（ID: 804643）
- 库迪咖啡（ID: 826079）
- 星巴克（ID: 39010）
- 蜜雪冰城（ID: 805164）

### 支持的渠道

- 抖音（平台类型: 2）
- 小红书（平台类型: 1）
- 微博（平台类型: 3）

## 数据输出

抓取的数据会导出为Excel文件，文件名为：

```
千瓜数据_品牌_渠道_开始日期_to_结束日期.xlsx
```

例如：
```
千瓜数据_古茗_抖音_2026-03-01_to_2026-03-15.xlsx
```

## 注意事项

1. 需要设置有效的Cookie才能成功抓取数据
2. 抓取的数据量较大时，可能需要较长时间，请耐心等待
3. 若需要添加新品牌，可在 `BRAND_ID_MAP` 中添加品牌名称和对应的ID
4. 若需要修改API地址或其他配置，可在 `CONFIG` 字典中进行修改
